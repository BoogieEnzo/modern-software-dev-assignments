# Week9 Repo Trends Explorer

每日简报页面：展示 GitHub Top 3 趋势仓库（按 7 天累计增星）。

## Run

```bash
pip install -r requirements.txt
make run
```

Open: http://localhost:8000

## Test

```bash
pip install -r requirements.txt
make test
```

## Env

- `GITHUB_TOKEN` (optional, recommended to avoid low API rate limits)
