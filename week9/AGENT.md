# Week9 Agent 开发任务线

## 远程开发环境

**架构**: Linux 服务器 + SSH + Mac Chrome 测试

### 常用命令

- **启动服务**: `make run` → 监听 `0.0.0.0:8002`
- **获取服务器 IP**: `hostname -I | awk '{print $1}'`
- **终止端口**: `lsof -ti:8002 | xargs kill -9`
- **Mac 访问**: `http://<server-ip>:8002`
- **SSH Tunnel**: `ssh -L 8002:localhost:8002 fengde@<server-ip>`

### 注意事项

- 端口使用 **8002**（避免与 8000 冲突）
- 服务绑定 `0.0.0.0` 以支持远程访问
- 需要设置 `GITHUB_TOKEN` 避免 API 限流
